## ?? Design Patterns

### Elevation System
```
Level 0: Background     (no shadow)
Level 1: Surface        (no shadow)
Level 2: Cards          (soft shadow)
Level 3: Menus          (drop shadow)
Level 4: Modals         (strong shadow)
```

### Corner Radius Scale
```
Small:  4px   - Buttons, inputs
Medium: 8px   - Cards, containers
Large:  10px  - Badges
XLarge: 12px  - Modals, overlays
```

### Spacing Scale
```
xs:  4px    - Tight spacing
sm:  8px    - Related items
md:  12px   - Standard gap
lg:  16px   - Section padding
xl:  20px   - Major sections
xxl: 24px   - Large gaps
```

### Font Size Scale
```
xs:  12px   - Small labels
sm:  13px   - Secondary text
md:  14px   - Body text
lg:  16px   - Emphasis
xl:  18px   - Subheadings
xxl: 24px   - Headings
```
